# ScrollBar

當初不喜歡滾動條的樣式

所以想製作一個來取代

最初是使用 jQuery 完成

但覺得不要過度依賴 jQuery

改使用原生 js 完成 所有功能

花了一些時間實作了 drag 的功能

和了解了幾個 event

MutationObserver  觀察子元素是否有變動時

Touch Event 觸控事件處理

wheel 滾動事件
